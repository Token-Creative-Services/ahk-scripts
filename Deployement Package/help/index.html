Timelapse Help

This documenation will guide any rep on the use and practices of running Timelapse scirpts for social media RPA. Scripts are organized by tool category, and then contain options/inputs for bulk-account use. Most scripts run with hot-keys, so be sure to familiarize yourself with all of them!

WARNING: These scripts are very powerful, and if run incorrectly will result in account(s) easily getting banned. Start SLOW!!

Getting Started

There are several scripts that will run based on pre-installed and configured apps. Be sure you download and prepare all apps prior to running any scripts! We *HIGHLY* recommend ensuring both your Display Settings(Windows) & Browser Zoom levels are at 100% (neutral).Many scripts contain inputs to specify the # of accounts you wish to run in-bulk

AutoHotKey - Required for running all scripts. Download

Bluestacks - For running scripts on instagram, we opt to run through a VM app called Bluestakcs. You must down,install, and log into your Instagram app in Bluestacks in order to run Instagram Scripts! Download

CrowdFire - No download require for this one. Ensure you have accessed the CrowdFire app before running scripts, as you will need to be authenticated first. Access here

Deployement Package - To get the scripts running, download and deploy the scripts package to "C:\Program Files\". These be downloaded here

Instagram @ Bluestacks - IG running inside of the bluestacks VMware

Follow

Follows a series of new profiles from the top accounts of profiles you are already following.

Requirement: Bluestacks is open and running, with Instagram app in focus, maximized.

Input:The number of accounts logged into instagram (max 5).

HotKeys: [~] to Exit the app. | [TAB] to Pause

Utilities

Utilities All purpose scripts designed for every day life

X/Y Finder

This script is run and toggled with the [CAPS-LOCK] key. When enabled, it will display the X/Y cooridantes of your current mouse position. Press [PRINT SCREEN] to save to clipboard

Get Colour

When [Control]+[Alt]+[Z] are pressed, the colour beneath the mouse's current location is pulled. Displays Hex RGB colour code

Auto-Clicker

When running, script is toggled with [F6] key to click on the current mouse location at a high interval.
